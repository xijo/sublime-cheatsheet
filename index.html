Sublime Text 101

Basics
ctrl + c Copy current selection or line
ctrl + v Paste clipboard to current position
ctrl + x Cut current selection or line
ctrl + n Open new file
ctrl + s Save current file
ctrl + f Find in current file
ctrl + shift + f Find in all files
Movement
ctrl + ← → Jump between words
pos1 / end Jump between start and end of line
Selecting text
ctrl + d Select word / select more occurrences
ctrl + l Select whole line
ctrl + shift + ↑ ↓ Move selection up or down
ctrl + alt + ↑ ↓ Open multiple carets
